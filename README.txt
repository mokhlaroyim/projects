#What stands for what:
- main.py -> main python file (run this file)
- python.py -> python file which is attached to main 
- haarcascade_frontalface_default.xml - xml file for face recognition 
- 4 images: black_widow.png, dr_strange.png, thor.png, woman_1.png
- PPT file 
- REPORT file


#HOW TO RUN PROGRAM:
1. Run main.py file -> you will get Window popup named "Menu"
2. Click on one of the Images -> Camera pops up
3. Loot at the Camera, so that your face can be clearly captured by camera. 
4. Press on Space to make a picture 
5. Press on Escape and get the output
